O repositório contém o código-fonte da aplicação React + Vite, mas o build estático (dist/) não está publicado.

Para publicar corretamente no GitHub Pages, gere o build e copie o conteúdo de dist/ para a pasta docs/ ou publique a branch gh-pages.

Passos rápidos (no seu computador)

  1. Instale dependências: npm install
  2. Gere o build: npm run build
  3. Copie o conteúdo: rm -rf docs && cp -r dist docs (ou use o Explorer/Drag & Drop)
  4. Commit e push: git add docs && git commit -m "Deploy: build to docs" && git push
  5. No GitHub: Settings → Pages → Source: main / /docs

Se preferir, gere o build localmente e envie apenas o conteúdo de dist/ via Add file → Upload files no GitHub (arraste os arquivos do dist, não envie o .zip).
